Privacy Policy — Quản Trị Page

Effective Date: July 4, 2026 | Last Updated: July 4, 2026 | Version: 1.0

1. Overview

"Quản Trị Page" (App ID: 1477228617423402, Business type) is a content management utility for Facebook Pages, developed and operated by Phúc Lâm (individual).

This privacy policy explains what data is collected when you use the app, how that data is used and protected, and your rights regarding your data. The app is built on the principle of data minimization — collecting only what is strictly necessary to perform its core function (scheduling and publishing posts to your Facebook Pages).

2. Data We Collect

2.1 When you log in with Facebook

Data Purpose Required
Page Access Tokens Allow the app to call Graph API on your behalf to publish/schedule posts. ✅ Yes
List of Pages (id, name, category) Display pages you manage for publishing selection. ✅ Yes
Facebook name + avatar Verify admin identity. ✅ Yes

This data is transmitted directly from Meta to your device via the standard Meta OAuth flow, scoped only to permissions you explicitly approve. The app does not intercept or transmit this data to other systems.

2.2 When using the app

Data Description Storage location
Caption text The text content of your posts. Local Markdown files
Images / Videos Media files you upload to post. Local files in drafts directory
Schedule metadata Publish time and status (draft/scheduled/posted). YAML frontmatter in local Markdown files
Facebook post IDs Post IDs returned by Meta after successful publishing. Local tracking logs

2.3 Technical Logs

  • API errors: error responses from Meta API (status code + message) used for troubleshooting.
  • Response times: used locally for performance debugging.
  • Local IP: only used for local rate-limit protection, never transmitted elsewhere.

2.4 What we do NOT collect

  • Personal information (PII) of your Page followers or audiences.
  • Private messages on Facebook.
  • Comments or user-generated responses on your published posts.
  • Cookies, browser fingerprints, device IDs, or precise GPS location data.

3. How We Use Your Data

Purpose Data Needed Activity
Schedule & publish posts Token + caption + media + schedule Call Graph API at scheduled times
Manage multiple Pages Pages list + IDs Display in local app UI
Analyze performance Reach, engagement (from Meta API) Display metrics to the admin
Technical support Error logs Debug failed API calls

Your data is NOT used to: train AI/ML models, sell/rent to third parties, or construct behavioral advertising profiles.

4. Sharing with Third Parties

Your data is shared exclusively with Meta Platforms (Facebook, Inc.) via the Graph API within the scope of permissions you grant. We do not use third-party analytics SDKs (e.g., Google Analytics, Mixpanel) or sell data to advertisers.

5. Data Retention and Security

There is no cloud database. All data is stored locally on the admin's personal device. Tokens are stored in a local .env file, and post drafts are kept in a local folder. Technical logs are saved in temporary local directories and deleted automatically after 24 hours.

6. Your Rights

  • Revoke Access: Instantly via Meta Settings → Business Integrations.
  • Request Deletion: Email us at lamdo9@icloud.com. Request will be processed within 7 days.
  • Edit Information: You can directly modify any local files on your machine.

7. Contact

For any questions or data requests, contact us at:

  • Email: lamdo9@icloud.com
  • Owner: Phúc Lâm
